# Setclapp test task

Тестовое приложение: таблица записей с поиском, добавлением, редактированием и удалением. Данные сохраняются в **localStorage** браузера.

**Деплой:** [setclapp-test-task.vercel.app](https://setclapp-test-task.vercel.app/)

## Стек

- React 19, TypeScript
- Vite 8
- Ant Design 6, SCSS Modules
- dayjs

## Требования

- Node.js 20+ (рекомендуется LTS)

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение откроется по адресу, который покажет Vite (обычно `http://localhost:5173`).

## Скрипты

| Команда | Описание |
|--------|----------|
| `npm run dev` | режим разработки с HMR |
| `npm run build` | проверка TypeScript и production-сборка |
| `npm run preview` | локальный просмотр собранного билда |
| `npm run lint` | ESLint |
